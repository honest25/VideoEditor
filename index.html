Mandatory Story Structure — Trailer/Promo Rhythm

HOOK (0–3 s)
Strongest visual + bass hit / impact SFX
No title card • Face or dramatic reveal centered in upper 2/3
BUILD (4–70% duration)
Visual change ≤ every 3–5 seconds
Emotional curve: curiosity → tension → connection
Use J-cuts (audio leads picture) for natural flow
PEAK (80–90%)
Biggest music drop + slowest / fastest moment sync
Wide-to-tight zoom or dramatic match cut
CTA / LANDING (last 10%)
Music resolve (major chord or fade)
Minimal text fade-in (3–5 words max) • Hold 2–3 s

Advanced Free Techniques in CapCut

J-cut / L-cut: Drag audio track longer/shorter than video → natural dialogue bleed
Match Cut: Cut on similar shape/motion (e.g. eye → sun) — manual split & align
Speed Ramp: Select clip → Speed → Curve → Add keyframes (0.5x slow → 2x push)
Film Grain + Light Leak: Import free 35mm grain PNG → Overlay blend mode: Overlay / Soft Light → Opacity 15–25%
Motion Blur Simulation: Effects → Basic → Motion Blur → Keyframe intensity on fast pans
Whoosh / Bass Reveal: Search CapCut SFX "whoosh cinematic" or import free bass hit → Sync to cut

Emotion-Driven Color + Audio Strategy

Color Grading (Free LUTs & Manual)
Adjust → Basic → Temperature: -5 to +10 (cool=tension, warm=nostalgia)
Contrast +15–30 • Shadow +10 • Highlight -10
Saturation -5 to -15 (cinematic desaturated look)
Add free "Cinematic" or "Teal & Orange" filter → Reduce intensity 40–60%
Vignette (Effects → Basic) → Strength 20–35 → Focus center face
Audio Design
Music: Search CapCut "Cinematic Emotional" / "Trailer Epic" (free library)
Curve: Start low → Build → Peak drop → Resolve fade
Dialogue: Extract → Noise Reduction (Audio tab) → Volume normalize -6dB LUFS
Layers: Ambient bed (-18dB) + SFX (-12dB) + Music master (-9 to -12dB peak)
Typography Rules
Font: "Montserrat Bold" or "Bebas Neue" (free in CapCut)
Size: 8–12% of height • Center safe zone (upper 60%)
Animation: Fade in + subtle scale-up (keyframes) only on emotional words

Example: 15-Second Netflix-Style Promo Edit Plan

HOOK (0–3s)
• Clip: Dramatic close-up eye open + whoosh SFX
• Music: Low drone swell → bass hit at 2.8s
• Grading: Cool Teal push, heavy vignette

BUILD (3–10s)
• J-cut dialogue: "Everything changes tonight..." (audio starts 0.4s early)
• 4 quick cuts: wide landscape → running feet → gun cock → face reveal
• Speed ramp 80% → 40% on gun cock
• Match cut: gun barrel → tunnel light

PEAK (10–13s)
• Slow-mo reveal (0.4x) + music drop
• Overlay light leak + film grain 25%
• Text: "COMING 2026" (fade + scale 110%)

CTA (13–15s)
• Music resolve to soft pad
• Text: "Watch Now • Link in Bio" (hold 2s, center safe)
• Fade to black 0.6s

Platform Notes: Export 1080p 60fps H.264 • Bitrate High • Keep under 100 MB for fast upload
