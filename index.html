Mandatory Story Structure — Trailer Rhythm

HOOK (0–3s)
Immediate trigger: strongest visual + bass/impact SFX sync • Center face/reveal upper 2/3 • No logo/text
BUILD (4–70%)
Visual refresh every 3–5s • Curiosity → tension rise → connection • Heavy J-cuts (audio leads picture)
PEAK (80–90%)
Strongest music beat/drop + visual climax (slow-mo reveal/match cut) • Layer grain/leak for drama
CTA / LANDING (last 10%)
Music resolve (soft pad/major fade) • Minimal text (3–5 words) fade-in hold 2–3s • Smooth 0.6–1s fade to black

Advanced Free Techniques in VN

J-cut / L-cut: Drag audio track edges beyond video clip (audio leads/trails picture)
Match Cut: Split clips → align similar shape/motion (eye→light flare, gun→tunnel)
Speed Ramp: Clip → Speed → Curve → Add keyframes (e.g. 1x → 0.4x slow → 1.8x push)
Film Grain + Light Leak: Import free PNG/MP4 → overlay track → Blend: Overlay or Soft Light • Opacity 15–30% • Keyframe fade
Motion Blur Sim: Effects → Blur → Motion Blur → Keyframe intensity on fast pans only
Whoosh / Bass Hit: VN SFX "cinematic whoosh" / "bass impact" or import → snap to transition/reveal points
